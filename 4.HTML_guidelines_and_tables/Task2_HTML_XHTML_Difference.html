Difference between HTML and XHTML
S.No HTML XHTML
1 HTML - Hypertext MarkUp Language XHTML - Extensible Hypertext MarkUp Language
2 It was Developed by Tim Berners Lee It was developed by W3C - World Wide Web Consortium
3 It was initially released in 1993 It was initially release on 26 Jan 2000
4 It is in Document file format It has MarkUp language as the file format
5 Filename Extension - .html,.htm Filename Extension - .xhtml,.xht,.html,.htm,.xml
6 It is extended from standard generalized MarkUp Language It is extended from HTML and XML
7 It is of the type text/html It is of the type application / XHTML+XML
8 HTML files are not case sensitive XHTML files are case sensitive. Everything must be in lower case.
9 All tags need not to be closed in order of opening. Open tags can be used in an HTML file All tags needs to be closed in an XHTML
10 HTML is less expressive in comparison of XHTML XHTML is more expressive with respect to HTML
11 HTML does not call for a mandatory root element XHTML present mandatory root element to be present.
12 All content can be included under the body element Content must be placed in the form of blocks
